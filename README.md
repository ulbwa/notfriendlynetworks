# notfriendlynetworks

Обновляемые списки подсетей и доменов недружественных государств. Репозиторий предназначен для блокировки вредоносных сайтов и защиты от мошенников в интернете

Каждый набор правил — sing-box rule-set (`ruleset.json`, version 2) с доменами, подсетями и Android-пакетами, файлом доступности по регионам (`config.json`) и описанием (`README.md`).

**Обозначения доступности:** ✅ доступен напрямую · 🌐 гео-ограничение сервиса · ⛔ заблокирован государством. Колонки: 🇷🇺 Россия · 🇺🇦 Украина · 🇰🇿 Казахстан · 🇧🇾 Беларусь · 🌍 остальной мир.
